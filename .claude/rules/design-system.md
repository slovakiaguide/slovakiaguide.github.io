# Дизайн-система

## Заголовки

- На странице должен быть ровно один `h1`. Лэйаут `page` автоматически рендерит `page.title` как `<h1 class="page-title">`.
- Иерархия заголовков в контенте начинается с `h2`.

## Структура страницы

Лэйаут `page` оборачивает контент в `.page-content`. Навигация — `.left-sidebar`, оглавление — `.right-sidebar`.
